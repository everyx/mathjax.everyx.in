使用说明

为了在能在 github markdown 文件中插入 amthjax 生成的图片而提供的一个在线转换工具。

注意：PNG 格式的暂时不能用，主机没装 java 环境，而我又没权限装，ㄟ( ▔, ▔ )ㄏ

URL 规则

http://[表达式类型]/[输出形式]/[表达式]

表达式: 经过 base64 编码的表达式。

表达式类型: am(AsciiMath), mml(MathML), page(HTML5 页面), tex(TeX or LaTeX)。

输出形式：png, svg, mml(MathML)。
